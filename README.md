# Expense Tracker – Full Stack Application

A beginner-friendly **Full Stack Expense Tracker** application built using  
**Spring Boot (Backend)** and **React.js (Frontend)** with **MySQL** database.

This project helps users add, view, and manage their daily expenses with monthly and category-wise insights.



## => Features

- Add expenses (title, amount, category, date)
- View all expenses in a list
- Monthly expense summary
- Category-wise expense tracking
- Simple and clean UI
- REST API based architecture



## => Tech Stack

### Backend
- Java
- Spring Boot
- REST APIs
- MySQL
- Maven

### Frontend
- React.js
- Axios
- HTML, CSS
- JavaScript

## => How to Run the Project

### Backend (Spring Boot)
1. Open `expense-tracker-backend` in IntelliJ / Eclipse
2. Configure MySQL in `application.properties`
3. Run the Spring Boot application
Backend runs at: http://localhost:8080



### Frontend (React)
```bash
cd expense-tracker-frontend
npm install
npm start
Frontend runs at: http://localhost:3000
